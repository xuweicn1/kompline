
## 采样设备

- 砂轮重量：896克
- 平衡仪传感器：3个HX711压力传感器，间隔120度，均匀分布圆周
- 支撑点圆直径：366mm，上面加支架，支撑位采用钢珠



## 数据源：

取一片砂轮，手动置于平衡仪上：

1. 读取3个传感器数据记为：sensor_0,sensor_1,sensor_2；
2. 估算对应角度记为：angle

拿起砂轮旋转约20度再次放置于平衡仪上，同上取数，重复若干次，得到数据集

数据集如下：
